# AI-Enhanced IoT Health Monitoring System

## Overview
This project monitors patient health data and uses machine learning to detect abnormal conditions such as high heart rate or temperature.

## Features
- Real-time health data monitoring  
- Machine learning-based anomaly detection  
- Alert system for abnormal conditions  

## Tech Stack
- Python  
- Machine Learning  
- NumPy, Pandas  

## Workflow
1. Collect health data (heart rate, temperature)
2. Preprocess data
3. Train machine learning model
4. Detect abnormal health conditions

## How to Run
1. Clone the repository  
2. Install required libraries  
3. Run the Python file  

## Future Improvements
- Integration with IoT devices  
- Mobile app alerts  
- Real-time dashboard
